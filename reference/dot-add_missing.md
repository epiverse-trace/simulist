# Introduce user-specified proportion of custom missing values into a `<data.frame>`

Introduce user-specified proportion of custom missing values into a
`<data.frame>`

## Usage

``` r
.add_missing(linelist, .args)
```

## Arguments

- linelist:

  Line list `<data.frame>` output from
  [`sim_linelist()`](https://epiverse-trace.github.io/simulist/reference/sim_linelist.md).

- .args:

  A list of setting from
  [`messy_linelist()`](https://epiverse-trace.github.io/simulist/reference/messy_linelist.md).

## Value

A line list `<data.frame>`
