# Package index

## Simulation functions

Functions to simulate disease outbreak data.

- [`sim_contacts()`](https://epiverse-trace.github.io/simulist/dev/reference/sim_contacts.md)
  : Simulate contacts for an infectious disease outbreak
- [`sim_linelist()`](https://epiverse-trace.github.io/simulist/dev/reference/sim_linelist.md)
  : Simulate a line list
- [`sim_outbreak()`](https://epiverse-trace.github.io/simulist/dev/reference/sim_outbreak.md)
  : Simulate a line list and a contacts table

## Right-truncate line list

- [`messy_linelist()`](https://epiverse-trace.github.io/simulist/dev/reference/messy_linelist.md)

## Censor line list
